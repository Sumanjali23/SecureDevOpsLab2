## 📌 Overview

SecureDevOpsLab2 is the second in a series of **Secure DevOps labs**, focused on building and deploying a containerised full-stack application while analysing and mitigating common security vulnerabilities in web applications.

---

## ✨ Features

- 🐍 Flask REST API backend
- ⚛️ React TypeScript frontend
- 🐳 Docker Compose multi-container setup
- 🔐 Security analysis and hardening exercises

---

## 🚀 Getting Started

### Prerequisites
- Docker & Docker Compose

### Run the Application
```bash
docker-compose up --build
```

### Services
| Service | Port |
|---|---|
| Frontend (React) | `http://localhost:3000` |
| Backend (Flask) | `http://localhost:4000` |


📦 SecureDevOpsLab2
┣ 📂 backend-flask/       # Flask REST API
┣ 📂 frontend-react/      # React TypeScript UI
┣ 📄 docker-compose.yml   # Service orchestration
